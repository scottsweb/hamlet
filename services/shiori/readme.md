# Shiori

[Shiori](https://github.com/go-shiori/shiori) is a simple bookmark manager built with Go. It's able to import and export browser compatible bookmark files and create offline archives of important links. 

## .env

Add a random `SHIORI_HTTP_SECRET_KEY`.

```
PUID=1000
PGID=1000
SHIORI_HTTP_SECRET_KEY=!password
```
